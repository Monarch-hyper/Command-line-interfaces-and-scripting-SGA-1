# Linux Environment Verification

## Objective

The objective of this task is to verify the Linux working environment using basic command-line utilities.

## Commands Used

### `whoami`
Displays the currently logged-in username.

### `groups`
Shows the groups to which the current user belongs.

### `echo $SHELL`
Displays the default shell currently in use.

### `pwd`
Prints the current working directory.

### `ls -la`
Lists all files and directories, including hidden files, to inspect the user's home directory.

### `ping -c 4 google.com`
Tests internet connectivity by sending four ICMP packets to Google's server.
